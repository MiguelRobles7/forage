# Forage

Major Course Output for CCAPDEV (Web Application Development) course at De La Salle University.

A Food Review App

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Developing

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```
